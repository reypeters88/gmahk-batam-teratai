Aplikasi HP & Tablet

Instal GMAHK TERATAI

Progressive Web App (PWA)

Instal Aplikasi di HP & Tablet

Akses cepat dari Homescreen tanpa perlu buka browser berulang kali

Instal Langsung ke Perangkat

Tekan tombol di bawah untuk otomatis menginstal aplikasi ini.

Pilih Panduan Sesuai Perangkat Anda:

1

Buka aplikasi ini di Google Chrome atau Microsoft Edge

Pastikan URL aplikasi sudah terbuka sepenuhnya di browser HP Android Anda.

2

Ketuk ikon Menu Tiga Titik ⋮ di pojok kanan atas

Akan muncul menu pilihan pengaturan browser Chrome/Edge.

3

Pilih "Tambahkan ke Layar Utama" atau "Instal Aplikasi"

Konfirmasi dengan menekan tombol Tambahkan / Instal. Ikon GMAHK TERATAI akan langsung muncul di Homescreen HP Anda!

1

Buka aplikasi melalui browser Safari di iPhone / iPad

Fitur instalasi di iOS memerlukan browser Safari bawaan Apple.

2

Ketuk tombol Bagikan di bilah bawah Safari

Tombol Share (ikon kotak dengan panah ke atas) terletak di bagian tengah bawah layar.

3

Gulir ke bawah & pilih "Tambahkan ke Layar Utama" (Add to Home Screen)

Lalu ketuk tombol Tambah (Add) di pojok kanan atas. Aplikasi siap digunakan seperti aplikasi iPhone natif!

1

Tablet Android (Samsung Galaxy Tab / Xiaomi Pad / dll)

Tampilan aplikasi secara otomatis menyesuaikan layar lebar tablet (Responsive Split View). Klik ikon Instal di address bar Chrome atau pilih menu Tambahkan ke Layar Utama.

2

iPad (Apple Tablet)

Gunakan Safari -> ketuk ikon Share di pojok kanan atas -> pilih Add to Home Screen. Aplikasi mendukung mode layar penuh mendatar maupun tegak.

3

Komputer / Laptop (Windows & Mac)

Di Chrome / Edge, klik ikon komputer dengan tanda plus ⊕ di ujung kanan kolom alamat web untuk menginstal sebagai aplikasi Desktop.
